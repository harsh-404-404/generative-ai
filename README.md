This repository provides a simple and beginner-friendly introduction to the core tools used in Python programming and data analysis. It covers the fundamentals of Python, including variables, data types, loops, conditions, and functions, along with essential libraries like NumPy for numerical computation, Matplotlib for data visualization, and Pandas for handling and analyzing datasets. Each section includes clear examples that help students understand how to work with arrays, perform mathematical operations, create visual plots, and manipulate tabular data. Whether you're starting your programming journey or preparing for data-related projects, this repository offers a concise and practical foundation.
